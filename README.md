# Super Dupper Guitar App

This folder is ready for GitHub Pages.

## Files
- `index.html` — Metronome and Scale Viewer
- `circle.html` — Circle of Fifths

## Settings persistence
The site saves user preferences in the browser using `localStorage`, including theme and the main viewer settings, so they persist between pages and future visits.

## GitHub Pages
Upload the contents of this folder to the root of your GitHub repository, then enable GitHub Pages for the repo.
